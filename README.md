# Proyecto Terremotos

Proyecto end-to-end de análisis de datos sísmicos usando Python, MySQL, FastAPI, Streamlit, Docker y Git.

## Objetivo

Construir una solución de análisis de terremotos que permita extraer, transformar, cargar y visualizar información sísmica para apoyar decisiones técnicas y ejecutivas.

## Tecnologías

- Python
- Pandas
- MySQL
- SQLAlchemy
- FastAPI
- Streamlit
- Plotly
- Docker
- Git / GitHub
- Pytest

## Estructura

```text
api/            API REST con FastAPI
dashboards/     Dashboard interactivo Streamlit
data/           Datos raw y processed
etl/            Pipeline ETL
tests/          Pruebas automatizadas
docs/           Documentación técnica
